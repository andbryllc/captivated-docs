---
title: "Customize Website Widget (Text and Live Chat)"
description: "Admins can customize their website widget's color, shape, size, placement, text, icon, and live chat options."
---

Admins of Captivated have access to customize their website widget design such as color, shape, size, icon, text.

First, navigate to the Administration section of Captivated by hitting on the cog wheel in the bottom left-hand side of the screen. Then scroll down to the tile titled "Website Widgets"

![](https://learn.captivated.works/wp-content/uploads/2023/08/website-widget-new.png)

The widget will now show if it is currently tied to a disabled channel (phone number) in Captivated.

![](https://learn.captivated.works/wp-content/uploads/2023/08/disabled-widget.png)

Click into the widget you'd like to edit. On the Settings tab, you will see the channels that the widget is assigned to as well as the allowed domains.

If you have live chat enabled, you will see additional chat options that you can customize.

![](https://learn.captivated.works/wp-content/uploads/2023/08/settings-widget.png)

Under the Custom Text tab, you can customize the text that appears when opening the website widget. If you have live chat enabled, additional options will show.

![](https://learn.captivated.works/wp-content/uploads/2023/08/custom-text.png)

The Design tab allows you to customize:

1. Color
2. Shape
3. Size
4. Placement on website
5. Text
6. Icon
7. Live chat or Text Us button on top

![](https://learn.captivated.works/wp-content/uploads/2023/08/design.png)

Finally, the installation tab provide all necessary instructions to add the widget to your website. Additionally, if you are using [Google Tag Manager](/integrations/google-tag-manager), check out this additional page documentation.

![](https://learn.captivated.works/wp-content/uploads/2023/08/installation-widget.png)

If your organization utilizes our live chat, you can additionally customize these items:

- Specific hours where the live chat is open or closed. (Great to ensure you have staff to cover). To set this up, create a [queue](/administration/admin-queues) specifically for Live Chat.
- Departments or Location dropdown. We can set up a dropdown so your potential clients can be routed to the proper department or location.
