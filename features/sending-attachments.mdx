---
title: "Sending Attachments"
description: "Send photos, videos, and documents via text in Captivated."
---

You can send several types of attachments in Captivated including photos (.png, .jpeg, etc.), videos, PDFs, Word and Excel files. To send an attachment, select the paperclip icon underneath the message field. Choose your attachment, add a message along with the file if desired, then send your message.

![](https://learn.captivated.works/wp-content/uploads/2018/01/Screen-Shot-2018-12-01-at-5.13.54-PM-1024x734.png)

You can also remove an attached item by clicking the red "X" icon right next to the attachment's name.

![](https://learn.captivated.works/wp-content/uploads/2018/01/Screenshot-2025-08-19-at-11.05.51%20AM.png)

MMS (images, videos and audio) have a size limit per carrier. Reduce your file size to ensure successful deliverability.

![](https://learn.captivated.works/wp-content/uploads/2018/01/Screenshot-2025-08-19-at-11.10.12%20AM.png)

To check your file size, download to your computer and hit info and it will show you the size of the file. If you need to reduce your file size, there are plenty of free services available on the internet to use. Here is what it looks like using Mac.

![](https://learn.captivated.works/wp-content/uploads/2018/01/Screenshot-2025-08-19-at-11.12.25%20AM.png)

Questions? You can chat with Captivated's support team using the "Support" option in the left-hand menu of our app.
