## Step by Step Instructions

To message a new contact, select the green new message button in the top right corner of your screen.

![New message button](https://learn.captivated.works/wp-content/uploads/2018/01/Screenshot-2023-08-29-at-3.38.38-PM.png)

You can search for a contact by name or phone number.

![Search for a contact](https://learn.captivated.works/wp-content/uploads/2018/01/Screenshot-2023-08-29-at-3.40.01-PM.png)

*If the contact already exists within Captivated, you'll be able to select the contact as an option and message them*

![Select an existing contact](https://learn.captivated.works/wp-content/uploads/2018/01/Screenshot-2023-08-29-at-3.41.03-PM.png)

*If the contact does not yet exist, you'll have the option to "+Add a new contact."*

Add your contact's first and last name, then select "Save Contact and Continue".

Optionally, you can also input a salutation the "Salutation" field which works with your [templates](/features/templates). Using \{\{salutation\}\} within a template will pull whatever is in the "Salutation" field of a contact's profile. If the field is left blank, it will pull from the "First Name" field.

![Add a new contact](https://learn.captivated.works/wp-content/uploads/2018/01/Screenshot-2023-08-29-at-3.44.10-PM.png)

Note: Once you've selected your recipient, you might have the option to select the [SMS channel](/administration/admin-channels) you would like to send the message from (only applies if you have access to multiple channels).

![Select an SMS channel](https://learn.captivated.works/wp-content/uploads/2018/01/Screenshot-2023-08-29-at-3.47.03-PM.png)

Once you've either selected an existing contact or saved a new contact, you can draft your message in the message box and hit the green "Send" button to the right of your message field to send your message.

![Draft and send your message](https://learn.captivated.works/wp-content/uploads/2018/01/Screenshot-2023-08-29-at-3.50.18-PM.png)
