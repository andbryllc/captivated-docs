---
title: "SMS Failed Message Indicator"
description: "Use Captivated's failed message indicator to view the error code and reason behind a failed text message."
---

If your text message to a contact displays a 'Message failed' icon above the message, you can click on the down arrow to the far right of the icon to gain more insight.

![](https://learn.captivated.works/wp-content/uploads/2023/12/Blog-5-2.png)

Clicking on 'View Error' in the dropdown will display the error message code along with the reason your message has failed.

![](https://learn.captivated.works/wp-content/uploads/2023/12/Blog-5-1.png)

![](https://learn.captivated.works/wp-content/uploads/2023/12/Blog-6.png)

![](https://learn.captivated.works/wp-content/uploads/2023/12/Blog-3.png)
