## Step by Step Instructions

![Secure Chat by Captivated](https://learn.captivated.works/wp-content/uploads/2020/05/Secure-Chat-by-Captivated-473x1024.jpg)

To start a chatting, you can text an invitation for a secure chat. Hit the three dot menu and then select "invite to chat". Use it from your desktop browser or your smartphone with our mobile app.

![Invite to chat menu](https://learn.captivated.works/wp-content/uploads/2023/10/Screenshot-2023-10-11-at-2.07.45-PM.png)

![Secure chat invite](https://learn.captivated.works/wp-content/uploads/2020/05/Secure-Chat-invite-Screeenshot-759x1024.jpeg)

**No software is required for your client/student/patient/customer to download.** They just click on the secure link and enter a 6-digit code sent to their phone via text to start a secure chat with your business.

![Access code screen](https://learn.captivated.works/wp-content/uploads/2020/05/Access-Code-Screenshot-471x1024.jpeg)

Once a secure chat is completed by the Captivated user, the conversation is **no longer accessible** from the recipient's mobile browser.

![Chat ended screen](https://learn.captivated.works/wp-content/uploads/2020/05/Chat-ended-screenshot-473x1024.png)

For security purposes, on a secure chat channel, the entire thread or individual messages can be **permanently** deleted, keeping the shared information secure**.**

![Delete secure messages](https://learn.captivated.works/wp-content/uploads/2023/10/Screenshot-2023-10-11-at-2.12.33-PM.png)
